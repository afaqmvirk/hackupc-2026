# Creative Swarm Copilot Web

Next.js App Router dashboard for the Smadex Creative Swarm Copilot MVP.

```bash
pnpm dev
pnpm test
pnpm lint
pnpm build
pnpm seed
```

`GEMINI_API_KEY` is required for swarm analysis, uploaded-image vision extraction, and Copilot Q&A. `MONGODB_URI` is optional; without it the app reads the bundled Smadex CSV dataset directly.
